## 布线规则

| 网络类型 | 线宽 | 说明 |
|----------|:----:|------|
| +48V 主电源 | **3.0mm** | 顶层, 到保险丝 |
| +12V 母线 | **1.5mm** | 顶层 |
| +5V 母线 | **1.0mm** | 顶层 |
| MOSFET D-S (灯路) | **1.0mm** | 顶层 |
| 信号线 (GPIO) | **0.3mm** | 顶层优先, 底层辅助 |
| GND | 铺铜 | 底层完整铜皮 + 顶层周边 |

## 布局分区

```
┌──────────────────────────────────────┐
│ [J1] [F1] [U1 XL7015]  [J2]         │ ← 高压区 (48V/12V)
│ ─────────────────────────────────── │
│        [U2 LM2596]                   │ ← 5V转换
│ ─────────────────────────────────── │
│   [J3雷达] [J4蜂鸣] [J6 C3] [J5开关]│ ← 信号外设区
│                                      │
│ ┌────────────────────┐              │
│ │ ESP32 38PIN 排母   │  [J7转向灯]  │
│ └────────────────────┘              │
│                                      │
│ [Q1] [Q2]  [D1 D2] [R..]           │ ← 功率区
└──────────────────────────────────────┘
```

## 制作步骤

1. 打开 KiCad 8.0 → `kicad/ebike-bsd.kicad_pro`
2. 打开原理图 → 检查网络连接
3. 打开 PCB → 布局 → 自动布线
4. DRC 检查 → Gerber 导出
5. 发嘉立创打样
