## 1. 系统简介

### 你装了什么

| 功能 | 是什么 | 有什么用 |
|------|--------|---------|
| **转向灯** | 前后左右 4 个 LED 灯条 | 告诉后车你往哪转 |
| **BSD 盲区监测** | 毫米波雷达 + 红色指示灯 | 侧后方有车时 LED 亮，提醒你 |
| **RCW 后碰撞预警** | 雷达 + 蜂鸣器 | 后方车快速逼近时蜂鸣大叫 |
| **转向辅助** | 雷达 + 指示灯 + 蜂鸣器 | 打转向灯时自动判断该方向安不安全 |

### 核心器件

```
[48V 电瓶] → [XL7015降压12V] → [LM2596降压5V] → [ESP32 主控]
                                                      ├── [MS60-3015 雷达]
                                                      ├── [MOSFET ×4] → [4个转向灯]
                                                      ├── [2个红色指示灯]
                                                      └── [蜂鸣器]
```

### 雷达覆盖范围

```
        左后方 ← ← ← ← ← → → → → 右后方
        ──────────────────────────────────
盲区:    [-40°════-15°] [正后] [+15°════+40°]
碰撞:    [-40°════════-5°] [+5°════════+40°]
         ← 有车 LED亮       有车 LED亮 →
```

- 正后方 ±14° 不报警（你能从后视镜看到）
- 最大探测距离：汽车 50m，实测有效 ~30m
- 不检测远离你的车

## 3. 安装步骤

### 3.1 安装前检查

1. 关掉电瓶电源
2. 万用表确认电瓶电压（正常 48-54V）
3. 所有器件摊开，对照清单清点

### 3.2 雷达焊接

雷达模块引脚很密（1.27mm 间距），**务必小心**：

```
MS60-3015 引脚定义（天线面朝上，邮票孔朝自己）：
 ┌─────────────────┐
 │  ██████████████  │ ← 天线面
 │  ██ MS60-3015 ██ │
 │  ██████████████  │
 │ ①  ②  ③  ④  ⑤  │ ← 5个半圆焊盘
 └─────────────────┘
 3.3V GND OUT RX  TX
```

| 引脚 | 接 ESP32 | 说明 |
|:---:|---------|------|
| ① | 3V3 或 5V | 供电（两者都支持） |
| ② | GND | 地线 |
| ③ | 悬空 | 不用接 |
| ④ | **GPIO17** | 雷达接收（ESP32→雷达） |
| ⑤ | **GPIO16** | 雷达发送（雷达→ESP32） |

> ⚠️ 焊接要点：烙铁 320°C，每个焊点 1-2 秒。每个焊完的引脚套热缩管。焊完用万用表蜂鸣档测相邻引脚是否短路。

### 3.3 电源系统安装

```
电瓶 48V+ ──┬── 保险丝 5A ── XL7015 IN+
            │
电瓶 48V- ──┴────────────── XL7015 IN-

XL7015 OUT+ ── LM2596 IN+
XL7015 OUT- ── LM2596 IN-

LM2596 OUT+ ── ESP32 5V (VIN)
LM2596 OUT- ── ESP32 GND
```

> ⚠️ **XL7015 调试顺序**：先不接负载，万用表测输出调到 **12V**，再接后续电路。**LM2596 同理**，调到 **5V**。

### 3.4 转向灯安装

| 灯位 | 接 ESP32 | 驱动方式 |
|------|---------|---------|
| 前左 | GPIO4 (左前后并联) | IRLZ44N MOSFET |
| 前右 | GPIO23 | IRLZ44N MOSFET |
| 后左 | — (已合并到GPIO4) | IRLZ44N MOSFET |
| 后右 | — (已合并到GPIO23) | IRLZ44N MOSFET |

MOSFET 接线：
```
ESP32 GPIO ── 100Ω 电阻 ── IRLZ44N G (栅极)
转向灯+ ──── 12V 电源
转向灯- ──── IRLZ44N D (漏极)
IRLZ44N S (源极) ──── GND
```

### 3.5 开关安装

| 开关 | 接 ESP32 | 说明 |
|------|---------|------|
| 左转 | GPIO13 | LOW=打左转 |
| 右转 | GPIO14 | LOW=打右转 |

把手三档开关（左-关-右），公共端 GND，左位接 GPIO13，右位接 GPIO14。GPIO27 预留（V2.6 已移除双闪功能）。

### 3.6 指示灯 + 蜂鸣器

| 器件 | 接 ESP32 | 说明 |
|------|---------|------|
| 左指示灯 + | GPIO25 → 220Ω → LED+ → GND | 刹车灯式安装 |
| 右指示灯 + | GPIO26 → 220Ω → LED+ → GND | 视野余光可见 |
| 蜂鸣器 I/O | GPIO12 | 高电平触发 |

### 3.7 雷达安装位置

- 安装高度：离地 **0.5-1.5m**
- 天线面（有丝印的一面）**正对后方**
- 前方无金属遮挡
- 用 3M 双面胶 + 扎带固定

## 5. 首次开机

### 5.1 开机

1. 打开电瓶开关
2. 听到 **蜂鸣器 3 声短鸣** → 系统正常启动
3. 雷达自动初始化（约 3 秒后进入工作状态）

### 5.2 静态测试（不上路）

| # | 操作 | 预期 |
|---|------|------|
| 1 | 开机 | 蜂鸣 3 声 |
| 2 | 打左转 | 前左+后左灯 1.5Hz 闪烁 |
| 3 | 打右转 | 前右+后右灯 1.5Hz 闪烁 |
| 5 | 站车后左侧 ~3m，挥手 | 仪表**左 LED 亮** + 蜂鸣一声 |
| 6 | 站车后右侧 ~3m，挥手 | 仪表**右 LED 亮** + 蜂鸣一声 |
| 7 | 关转向 | 灯灭 |

## 6. 灯和声音说明

### 指示灯（车把红色 LED ×2）

| 场景 | 左灯 | 右灯 | 声音 | 含义 |
|------|:---:|:---:|:----:|------|
| 后方无车 | 灭 | 灭 | 无 | 安全 |
| 左侧盲区有车 | 🔴 **慢闪 1Hz** | 灭 | 滴一声 | 侧方来车，保持直行 |
| 右侧盲区有车 | 灭 | 🔴 **慢闪 1Hz** | 滴一声 | 侧方来车，保持直行 |
| 打左转 + 左侧有车 | 🔴 **常亮** | 灭 | 🔊 **长鸣** | 别变道！ |
| 打右转 + 右侧有车 | 灭 | 🔴 **常亮** | 🔊 **长鸣** | 别变道！ |
| 后方车快速逼近（左） | 🔴 **快闪 4Hz** | 灭 | 🔊🔊🔊 **4Hz** | **立即加速/靠边！** |
| 后方车快速逼近（右） | 灭 | 🔴 **快闪 4Hz** | 🔊🔊🔊 **4Hz** | **立即加速/靠边！** |
| 后方双侧快速逼近 | 🔴 **快闪 4Hz** | 🔴 **快闪 4Hz** | 🔊🔊🔊 **4Hz** | **极度危险！** |

> 💡 三种灯效一眼区分：慢闪=盲区有车（提醒），常亮=打转向但不安全（警告），快闪=后车急逼（危险）

### 蜂鸣器声音速查

| 声音 | 对应灯效 | 持续多久 | 意思 |
|------|:---:|---------|------|
| "滴" 一声 | 慢闪 1Hz | 100ms | 有车进入盲区 |
| 4Hz 急促 | 快闪 4Hz | 车逼近期间 | **危险！后车快速逼近！** |
| 持续长鸣 | 常亮 | 关转向为止 | **你要转的方向有车！别变道！** |

### 紧急程度速查

| 系统 | 危急 | 灯 | 声音 | 你应该 |
|------|:---:|------|------|--------|
| 低速接近 | ℹ️ | 🔴 1Hz 慢闪 | 冷却5秒后一声 | 保持直行，等车过去 |
| 转向辅助 | 🟡 | 🔴 **常亮** | 长鸣 | **不要变道！** 等灯灭 |
| RCW 碰撞 | 🔴 | 🔴 **4Hz 快闪** | 4Hz 蜂鸣 | **立即加速或靠边！** |

## 7. 日常使用

### 7.1 骑行前检查

1. 开机听 3 声短鸣 = 系统正常
2. 试打左右转向灯
3. 看一眼指示灯是否无故亮着

### 7.2 骑行中

- **指示灯慢闪 1Hz** = 侧方盲区有车，保持直行
- **指示灯快闪 4Hz + 蜂鸣 4Hz** = 后方车逼近，检查后视镜
- **打灯 + 蜂鸣长鸣** = 别变道，后方有车
- **灯灭了** = 安全，可操作

### 7.3 停车后

- 关电瓶
- 检查接线是否松动（尤其是雷达和 MOSFET 插头）
- 雨天骑行后擦干雷达天线面

## 8. 故障排查

| 现象 | 可能原因 | 怎么修 |
|------|---------|--------|
| 开机无反应 | 保险丝烧了 | 换 5A 保险丝 |
| 开机不叫 | 蜂鸣器线松了 | 检查 GPIO12 |
| 转向灯不亮 | MOSFET 坏或线松 | 万用表测 GPIO 输出电压 |
| 指示灯一直亮 | 有车一直在盲区 / 雷达被遮挡 | 检查雷达天线面是否被泥遮住 |
| 指示灯一直不亮 | 雷达线松了 | 检查 GPIO16/17 和供电 |
| 蜂鸣器不叫 | 蜂鸣器线松 | 检查 GPIO12，万用表测电压 |
| 雨天乱报 | 雷达天线面有水珠 | 擦干 |

> **完整调试**：把 ESP32 的 USB 口接电脑，用串口工具 115200bps 看日志。

## 9. 安全须知

> ⚠️ **本系统是辅助设备，不能替代你自己的眼睛和判断。**
>
> 1. 变道前**必须回头看一眼**，不要只依赖指示灯
> 2. 雷达**不检测正后方的车**（±14° 范围内），后视镜是你的责任
> 3. 雷达**不检测远离你的车**
> 4. 下雨、泥巴可能遮挡雷达天线，导致探测距离变短
> 5. 第一次上路选择车少的路段测试
> 6. 电瓶改装涉及 48V 高压，**断电操作**，接线牢固
