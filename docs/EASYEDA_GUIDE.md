## 第2步：连线（完整网表）

连线从左到右、从上到下按顺序接：

### 2.1 电源部分

```
J1.1 (+)──── F1 ──── U1.1 (XL7015 IN+)
J1.2 (GND)── U1.2 (IN-) ── U1.4 (OUT-) ── 所有GND

U1.3 (OUT+ 12V) ──┬── J2.1 (12V输出+)
                    ├── U2.1 (LM2596 IN+)
                    ├── J7.3 (+12V去左灯)
                    └── J7.4 (+12V去右灯)

U2.3 (OUT+ 5V) ──┬── J3.1 (雷达VCC)
                   ├── J4.1 (蜂鸣器VCC)
                   ├── J6.4 (C3面板VCC)
                   └── ESP32 5V引脚 (排母对应脚)
```

### 2.2 转向灯驱动 (2路MOSFET)

> V2.7 每侧前+后灯在外部并联，板上只需2个MOSFET

```
ESP32 GPIO4 ──┬── R1(100Ω)── Q1.G(栅极)
               └── RPD1(10kΩ)── GND (下拉)

Q1.D(漏极) ── J7.1 (左前灯- + 左后灯- 在外部并联)
Q1.S(源极) ── GND

ESP32 GPIO23 ──┬── R2(100Ω)── Q2.G
                └── RPD2(10kΩ)── GND

Q2.D ── J7.2 (右前灯- + 右后灯- 在外部并联)
Q2.S ── GND
```

## 第3步：转为PCB

1. 顶部菜单 → **设计** → **转换为PCB**
2. 自动出现新标签页，元件散落

### 布局建议 (80×60mm)

```
┌──────────────────────────────────────────┐
│ [J1] [F1] [U1 XL7015]       [J2]         │ ← 高压区(48V/12V)
│ ──────────────────────────────────────── │
│          [U2 LM2596]                     │ ← 5V区
│ ──────────────────────────────────────── │
│ [J3雷达] [J4蜂鸣] [J6 C3]   [J5开关]   │ ← 信号区
│                                          │
│ ┌────────────────────┐  [J7转向灯]      │
│ │    ESP32 排母      │                  │
│ └────────────────────┘                  │
│                                          │
│ [Q1] [Q2]    [D1 D2]  [R..]            │ ← 功率区
└──────────────────────────────────────────┘
```

### 布线规则

| 网络 | 线宽 | 说明 |
|------|:----:|------|
| +48V | **3.0mm** | 到保险丝, 粗 |
| +12V | **1.5mm** | 到灯和LM2596 |
| +5V | **1.0mm** | 到ESP32/雷达/蜂鸣器 |
| 灯输出(D-S) | **1.0mm** | MOSFET→J7 |
| 信号(GPIO) | **0.3mm** | 细线即可 |
| GND | **铺铜** | 底层完整地平面 |

3. 布线完 → **检查DRC** → 绿色=通过
4. **导出 → Gerber** → 压缩zip → 上传嘉立创下单

---

## 重点注意事项

1. **GND必须共地** — 48V/12V/5V/MOSFET S极/LED负极全部连一起，底层铺完整铜皮
2. **ESP32插反=烧** — 排母方向注意：USB口朝向板边
3. **转向灯同侧并联** — 左前(-)+左后(-)一起接J7.1；右前(-)+右后(-)一起接J7.2；灯正极全部接+12V(J7.3/J7.4)
4. **MOSFET下拉电阻** — RPD1/RPD2各10kΩ接到GND，防止ESP32未初始化时栅极悬空导致灯误亮
5. **保险丝** — 放在48V输入端，紧挨J1，用3A慢熔
