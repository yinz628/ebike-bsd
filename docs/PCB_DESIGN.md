# e-Bike BSD 系统 PCB 设计规格书 V2.7

## 1. 板级参数

| 参数 | 值 |
|------|-----|
| PCB 尺寸 | 80mm × 60mm (双面板) |
| 铜厚 | 1oz |
| 板厚 | 1.6mm |
| 最小线宽 | 0.3mm (信号) |
| 最小间距 | 0.3mm |
| 表面处理 | HASL 有铅 |

## 2. 电路模块

```
┌─────────────────────────────────────────────┐
│  J1 48V输入  │ XL7015 48→12V │ 12V输出 J2   │
│  F1 保险丝   │   模块区域    │              │
├──────────────┴───────────────┴──────────────┤
│             LM2596 12→5V                    │
│             5V输出                          │
├─────────────────────────────────────────────┤
│  ┌──────────────────────────────────┐       │
│  │         ESP32-DevKitC            │       │
│  │         排母插座 (2×19)           │       │
│  └──────────────────────────────────┘       │
│                                             │
│  Q1-Q2: IRLZ44N ×2 (TO-220 卧装)            │
│  R1-R2: 100Ω 1/4W ×2                       │
│  RPD1-RPD2: 10kΩ 1/4W ×2                   │
│                                             │
│  J3: MS60-3015 雷达 (5pin 2.54排针)         │
│  J4: 蜂鸣器 (3pin)                          │
│  J5: 转向开关 (4pin)                        │
│  J6: C3显示面板 (4pin)                      │
│  J7: 转向灯输出 (4pin 5.08mm端子)           │
│  D1-D2: RCW LED (5mm 焊盘)                  │
└─────────────────────────────────────────────┘
```

## 3. 完整网络表 (V2.7)

### 电源网络

| 网络名 | 连接 |
|--------|------|
| `+48V` | J1.1 → F1.1 |
| `+48V_F` | F1.2 → XL7015 IN+ |
| `+12V` | XL7015 OUT+ → LM2596 IN+ → J2.1 → 转向灯阳极 |
| `+5V` | LM2596 OUT+ → ESP32 5V → J3.1 (雷达) → J4.1 (蜂鸣器) |
| `+3.3V` | ESP32 3V3 (板载) |
| `GND` | 所有模块共地 |

### 信号网络

| 网络名 | 起点 | 终点 | 说明 |
|--------|------|------|------|
| `RADAR_TX` | J3.5 (雷达TX) | ESP32 GPIO16 | 雷达→ESP32 |
| `RADAR_RX` | ESP32 GPIO17 | J3.4 (雷达RX) | ESP32→雷达 |
| `LED_LEFT_GATE` | ESP32 GPIO4 | R1→RPD1→Q1.G | 左转向MOSFET栅极 |
| `LED_RIGHT_GATE` | ESP32 GPIO23 | R2→RPD2→Q2.G | 右转向MOSFET栅极 |
| `LED_LEFT_OUT` | Q1.D | J7.1 (左灯-) | 左前+左后并联 |
| `LED_RIGHT_OUT` | Q2.D | J7.2 (右灯-) | 右前+右后并联 |
| `+12V_LAMP` | J2.1 | J7.3, J7.4 (灯阳极) | 转向灯共用12V |
| `BUZZER` | ESP32 GPIO12 | J4.3 (I/O) | 蜂鸣器信号 |
| `RCW_L` | ESP32 GPIO25 | R5→D1+ | 左RCW LED |
| `RCW_R` | ESP32 GPIO26 | R6→D2+ | 右RCW LED |
| `SW_LEFT` | ESP32 GPIO13 | J5.1→GND | 左转开关 (按下=低) |
| `SW_RIGHT` | ESP32 GPIO14 | J5.2→GND | 右转开关 (按下=低) |
| `U1_TX` | ESP32 GPIO18 | J6.2 (C3 RX) | C3显示面板发送 |
| `U1_RX` | ESP32 GPIO19 | J6.1 (C3 TX) | C3显示面板接收 |

### 转向灯接线 (外部)

| 端子 | 信号 | 说明 |
|------|------|------|
| J7.1 | LED_LEFT_OUT (Q1.D) | 左前+左后灯 负极 |
| J7.2 | LED_RIGHT_OUT (Q2.D) | 右前+右后灯 负极 |
| J7.3 | +12V | 所有转向灯正极共用 |
| J7.4 | +12V | 所有转向灯正极共用 |

> 同侧前后灯并联：左前灯(-) + 左后灯(-) → 一起接到 J7.1
> 右前灯(-) + 右后灯(-) → 一起接到 J7.2
> 所有灯的正极 → +12V 母线 (J7.3 或 J7.4)

## 5. 布局建议

```
顶层 (布线区):
  ┌──────────────────────────────────────┐
  │ [J1] [F1] [U1 XL7015]  [J2]         │ ← 高压区 (48V/12V)
  │ ─────────────────────────────────── │
  │        [U2 LM2596]                   │ ← 5V转换
  │ ─────────────────────────────────── │
  │   [J3雷达] [J4蜂鸣] [J6 C3]          │ ← 信号区
  │                                      │
  │ ┌────────────────────┐  [J5开关]    │
  │ │ ESP32 38P 排母     │              │
  │ └────────────────────┘  [J7转向灯]  │
  │                                      │
  │ [Q1] [Q2]    [D1 D2]  [R..]         │ ← 功率+指示区
  └──────────────────────────────────────┘
```

## 6. 布线规则

| 网络类型 | 线宽 | 说明 |
|----------|:----:|------|
| +48V 主电源 | **3.0mm** | 顶层, 到保险丝 |
| +12V 母线 | **1.5mm** | 顶层 |
| +5V 母线 | **1.0mm** | 顶层 |
| MOSFET D-S (灯路) | **1.0mm** | 顶层 |
| 信号线 (GPIO) | **0.3mm** | 顶层优先 |
| GND | 铺铜 | 底层完整铜皮 + 顶层周边 |

## 7. 制作步骤

1. 打开 [KiCad 8.0](https://www.kicad.org/download/) → 打开 `kicad/ebike-bsd.kicad_pro`
2. 检查原理图网络连接
3. PCB 布局 → 自动布线 → 手动调整
4. DRC 检查 → 制造输出 → Gerber
5. 提交嘉立创打样
