| 参数 | 值 |
|------|-----|

## 2. 电路模块

```
┌─────────────────────────────────────────────┐
│  J1 48V输入  │ XL7015 48→12V │ 12V输出 J2   │
│  F1 保险丝   │   模块区域    │              │
├──────────────┴───────────────┴──────────────┤
│             LM2596 12→5V                    │
│             5V输出                          │
├─────────────────────────────────────────────┤
│  ┌──────────────────────────────────┐       │
│  │         ESP32-DevKitC            │       │
│  │         排母插座 (2×15)           │       │
│  └──────────────────────────────────┘       │
│                                             │
│  Q1-Q2: IRLZ44N ×2 (TO-220 卧装)           │
│  R1-R2: 100Ω 1/4W ×2                       │
│                                             │
│  J3: MS60-3015 雷达 (5pin 2.54排针)         │
│  J4: 蜂鸣器 (3pin)                          │
│  J5: 转向开关 (4pin)                        │
│  J6: 转向灯输出 (4pin)                      │
│  D1-D2: RCW LED (5mm 焊盘)                  │
└─────────────────────────────────────────────┘
```

## 3. 完整网络表

### 电源网络

| 网络名 | 连接 |
|--------|------|
| `+48V` | J1.1 → F1.1 |
| `+48V_F` | F1.2 → XL7015 IN+ |
| `+12V` | XL7015 OUT+ → LM2596 IN+ → J2.1 → J6.1,3,5,7 |
| `+5V` | LM2596 OUT+ → ESP32 5V → J3.1 → J4.1 |
| `+3.3V` | ESP32 3V3 (板载) |
| `GND` | 所有模块的 GND 共地 |

### 信号网络

| 网络名 | 起点 | 终点 | 说明 |
|--------|------|------|------|
| `RADAR_TX` | J3.5 (雷达TX) | ESP32 GPIO16 | 雷达→ESP32 |
| `RADAR_RX` | ESP32 GPIO17 | J3.4 (雷达RX) | ESP32→雷达 |
| `LED_LEFT` | ESP32 GPIO4 | R1→Q1.G | 左侧转向灯 (前+后并联) |
| `LED_RIGHT` | ESP32 GPIO23 | R2→Q2.G | 右侧转向灯 (前+后并联) |
| `LED_L_OUT` | Q1.D | J6.1 | 接左侧灯负极 |
| `LED_R_OUT` | Q2.D | J6.3 | 接右侧灯负极 |
| `BUZZER` | ESP32 GPIO12 | J4.3 (I/O) | 蜂鸣器信号 |
| `RCW_L` | ESP32 GPIO25 | D1+ (串220Ω) | 左RCW LED |
| `RCW_R` | ESP32 GPIO26 | D2+ (串220Ω) | 右RCW LED |
| `SW_L` | ESP32 GPIO13 | J5.1→GND | 左转开关 |
| `SW_R` | ESP32 GPIO14 | J5.2→GND | 右转开关 |

> ℹ️ V2.6 起转向灯由 4 路 GPIO 合并为 2 路 (同侧前后并联), Q3/Q4 已不需要。GPIO27 预留 (双闪功能已移除)。

| U2 | LM2596 | 模块 | 1 | 12V→5V 2A, 模块 |

### MOSFET

| 编号 | 型号 | 封装 | 数量 | 说明 |
|------|------|------|:--:|------|
| Q1-Q2 | IRLZ44N | TO-220 | 2 | 逻辑电平 N-MOSFET (左/右各一, 同侧前后灯并联) |

### 电阻

| 编号 | 阻值 | 封装 | 数量 | 说明 |
|------|------|------|:--:|------|
| R1-R2 | 100Ω | 0805 或 1/4W 插件 | 2 | MOSFET 栅极限流 |
| R5-R6 | 220Ω | 0805 或 1/4W | 2 | RCW LED 限流 |

> ℹ️ 建议保留 R3/R4 + 10kΩ 栅源下拉电阻位 (Q1/Q2 各一个), 防止 ESP32 未初始化时 MOSFET 误通。

### 连接器

| J5 | 2.54mm 排针 | 3 | 转向开关 (L/R/GND) |

## 5. 布局建议

```
顶层:
  ┌──────────────────────────────────────┐
  │ [J1] [F1] [U1 XL7015]  [J2]         │ ← 上方: 高压区(48V/12V)
  │ ─────────────────────────────────── │
  │        [U2 LM2596]                   │ ← 中部: 5V转换
  │ ─────────────────────────────────── │
  │   [J3雷达] [J4蜂鸣]                  │
  │                                      │
  │ ┌────────────────────┐  [J5开关]    │ ← 左下: 信号区
  │ │ ESP32 排母插座      │              │
  │ └────────────────────┘               │
  │                                      │
│ [Q1] [Q2]    [J6]                    │ ← 右下: 功率区
│  D1   D2                            │
  └──────────────────────────────────────┘
```

## 6. 布线规则

- **高压区 (48V/12V):** 线宽 ≥1.5mm，与信号区间距 ≥3mm
- **功率区 (MOSFET):** D-S 走线 ≥2mm
- **信号区 (GPIO):** 0.3mm 即可
- **地线:** 顶层铺铜 GND，底层完整地平面
- **ESD:** 48V 输入加 TVS 管 (可选 P6KE51A)

## 7. 制作步骤

1. 打开 [easyeda.com](https://easyeda.com) 注册
2. 新建工程 → 新建原理图
3. 按上面的网络表放置元件、连线
4. 转换为 PCB → 按布局建议摆放
5. 自动布线 + 手工调整
6. 导出 Gerber → 发 JLCPCB 打样 (5片约 ¥20)
